# Portfolio Website

Source code of My porfolio website [4epiz.in](https://4epiz.in/)

## Tech Stack

**Client:** TailwindCSS

**Server:** Node, Express



## Runing the application

Clone the project

```bash
  git clone https://github.com/4Epiz/4epiz.in/
```

Go to the project directory

```bash
  cd 4epiz.in
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
